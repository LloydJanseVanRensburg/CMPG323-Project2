# Getting Started

!! You will need to have *AWS* account and *ParseBack4App* Account !!

`cd ./backend && npm i`
`cd ./frontend && npm i`

> Please make sure to add .env file in frontend root dir with flag
`SKIP_PREFLIGHT_CHECK=true`

### Env Variables Required In App Backend

- PORT=
- JWT_SECRET=
- PARSE_APPID=
- PARSE_JSAPIKEY=
- PARSE_MASTERKEY=
- PARSE_SERVERURL=
- AWS_REGION=
- AWS_ACCESS_KEY=
- AWS_SECRET_KEY=
- AWS_S3_BUCKET=
